# News-Week

> This project is a mock-up page of the Newsweek website.

![screenshot](./images/screenshot.png)

This project uses the bootstrap framework to build a replica of the newsweek site.

## Built With

- Bootstrap
- CSS
- HTML
- Favicon
- Font Awesome
